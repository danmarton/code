Value = Literal

A változó végső soron csak syntactic sugar, ami a numerikus memória-címet alfabetikus névvel helyettesíti.

Operátor = Függvény, csak prefix név helyett infix jellel hívjuk meg.

	Python:   int**int = hatványozás, str+str = konkatenáció, int*str = ismétlés

Magyar:
Literál ->  Változó  ->  Operátor/Függvény  ->  Kifejezés  ->  Utasítás
Angol:
Value  ->  Variable  ->  Operator/Function  ->  Expression  ->  Statement
